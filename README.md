# 🌱 Organic Feast

Organic Feast is a web-based organic food ordering platform developed using Django and Python. The project provides users with an easy and responsive interface for browsing and ordering organic food products online.

---

# 🚀 Features

- Organic product listing
- Product categories
- User authentication
- Shopping cart system
- Responsive UI
- Admin dashboard
- Database integration

---

# 🛠️ Tech Stack

- Python
- Django
- SQLite
- HTML5
- CSS3
- JavaScript
